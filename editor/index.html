Users and permissions

The content of the site will need to change over time and it is clear that users will benefit from being able to access the latest information. There is also a need to ensure each SMP reflects current evidence; supports CG’s in monitoring and reporting on delivery of SMPs.

Considerations:

  • How immediate do updates need to show?
  • How often will the data change?
  • How much of the data changes and needs to be directly editable?
  • Versioning and archiving
