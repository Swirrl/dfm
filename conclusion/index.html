Explore sources of data

Two possible sources of data have been identified which could either act as an initial (or potentially an ongoing) source of data.

  • The existing repository, which is an EAP file (Enterprise Architect), available as an export
  • The existing pipeline processes we discussed are all being run using FME

The next step will be to understand what is available from these sources, how complete they are and look at the best way to transform the data.

Users and permissions

A lot of choices around users and permissions will be dependent on other decisions, particularly on work to understand data sources. At its core the more granular control that is required over what users can do and the larger the number of them the more complexity is introduced and the longer the development time will be.
